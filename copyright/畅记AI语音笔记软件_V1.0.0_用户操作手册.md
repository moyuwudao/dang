## 2. 软件概述

### 2.1 软件简介

畅记是一款基于人工智能的语音笔记应用，旨在帮助用户快速记录、整理和分析语音内容。通过集成先进的语音转写技术和AI分析能力，畅记让笔记记录变得更加高效和智能。软件支持多种笔记类型，包括语音录音、文字速记、图片OCR识别等，并提供思维导图、智能周报、提醒管理等高级功能，全面提升个人知识管理效率。

### 2.2 主要功能

| 功能模块 | 功能描述 |
|---------|---------|
| 语音记录 | 支持高质量录音，实时显示波形 |
| 实时转写 | 将语音实时转换为文字 |
| AI分析 | 利用AI对笔记内容进行智能分析 |
| OCR识别 | 将图片中的文字提取出来 |
| 工具台 | 提供多种AI工具，如总结、待办事项、周报生成等 |
| 思维导图 | 可视化展示笔记关联，支持AI自动生成 |
| 提醒管理 | 创建和管理提醒事项，支持日历集成 |
| 智能周报 | AI自动汇总指定时间段的工作记录 |
| 速记 | 快速创建文字笔记 |
| 桌面小组件 | 在桌面添加录音、拍照、笔记快捷入口 |
| 订阅会员 | 套餐购买、余额充值、订单管理 |
| 数据管理 | 支持笔记的分类、搜索、导出、备份和恢复 |
| 回收站 | 已删除笔记的临时存放，支持恢复和永久删除 |
| API统计 | 查看API调用统计和用量分析 |
| 日志查看 | 查看应用运行日志，排查问题 |

### 2.3 运行环境

- **操作系统**：Android 8.0 及以上
- **存储空间**：至少 100MB 可用空间
- **网络要求**：需要联网使用AI转写和分析功能
- **权限要求**：录音权限、相机权限、存储权限、通知权限

## 4. 用户注册与登录

### 4.1 功能概述

畅记提供用户注册和登录功能，登录后可解锁云端AI服务、订阅套餐和余额充值等云端功能。未登录状态下，用户仍可使用本地录音、转写等基础功能。

### 4.2 手机号注册

**功能特点**：
- 支持中国大陆手机号注册
- 短信验证码验证
- 图形验证码防刷保护
- 注册即送100分钟体验额度

**操作步骤**：

1. 在登录页面点击"立即注册"
2. 输入11位手机号码
3. 点击"获取验证码"按钮
4. 输入收到的短信验证码
5. 设置密码（不少于6位）
6. 再次输入确认密码
7. 点击"注册"按钮完成注册
8. 系统弹出欢迎对话框，显示赠送的体验额度
9. 点击"开始使用"进入应用

【截图：注册页面，显示手机号输入框、验证码输入框、密码设置和注册按钮】

**注意事项**：
- 验证码发送后有60秒冷却时间
- 如触发风控，需输入图形验证码
- 密码长度不能少于6位
- 两次输入的密码必须一致

### 4.3 手机号登录

**功能特点**：
- 手机号+密码登录
- 密码可见性切换
- 忘记密码入口

**操作步骤**：

1. 在登录页面输入手机号
2. 输入密码
3. 点击密码框右侧的眼睛图标可切换密码可见性
4. 点击"登录"按钮
5. 登录成功后自动返回首页

【截图：登录页面，显示手机号输入框、密码输入框和登录按钮】

**注意事项**：
- 手机号需为11位数字
- 如忘记密码，点击"忘记密码?"链接进行重置

### 4.4 个人资料管理

**功能特点**：
- 查看用户基本信息
- 显示昵称和手机号
- 查看注册时间

**操作步骤**：

1. 进入设置页面
2. 点击用户头像或"个人信息"
3. 查看个人资料详情

【截图：个人信息页面，显示用户头像、昵称、手机号和注册时间】

## 9. 提醒管理

## 18. 常见问题解答

### Q1：转写功能需要联网吗？

A：是的，转写功能需要连接互联网，因为需要调用云端AI服务进行语音转文字处理。

### Q2：录音文件保存在哪里？

A：录音文件保存在应用的私有目录中，其他应用无法直接访问。

### Q3：如何备份数据？

A：在设置页面中找到"备份管理"选项，可以创建数据备份。支持选择备份内容类型和是否包含媒体文件。

### Q4：支持离线使用吗？

A：录音功能可以离线使用，但转写和AI分析需要联网。

### Q5：笔记支持分享吗？

A：支持，可以将笔记内容分享为文本或文件。

### Q6：删除的笔记能恢复吗？

A：删除的笔记会进入回收站，保留7天。在7天内可以恢复，超过7天或手动永久删除后无法恢复。

### Q7：如何使用思维导图功能？

A：进入"知识脑图"页面，系统会根据笔记标签自动生成脑图。也可以点击AI生成按钮，让AI智能分析笔记关联并生成脑图。

### Q8：周报生成支持自定义模板吗？

A：支持。在设置的分析模板中可以配置自定义周报模板，系统也提供标准周报、高管简报和详细周报三种内置模板。

### Q9：桌面小组件如何添加？

A：在Android桌面长按空白区域，选择"小组件"，找到畅记并拖动到桌面即可。小组件提供录音、拍照和速记三个快捷入口。

### Q10：API调用统计在哪里查看？

A：在"数据统计"页面中点击"API调用分析"，可以查看总调用次数、成功率、工具使用量和每日趋势等详细数据。

### Q11：注册账号有什么好处？

A：注册登录后可使用云端AI服务、购买订阅套餐、充值余额。新用户注册即送100分钟体验额度。

### Q12：如何查看应用运行日志？

A：在设置页面的"AI配置"标签页中，点击"运行日志"即可查看。支持按级别筛选和关键词搜索。
