# 畅记APP - 构建指南

## 📋 环境要求

### 必需软件
- **Flutter SDK** >= 3.0.0
- **Dart SDK** >= 3.0.0
- **Android Studio** 或 **IntelliJ IDEA**
- **Android SDK** (API 21+)
- **Git**

### 系统要求
- Windows 10/11、macOS 或 Linux
- 至少 4GB RAM（推荐 8GB）
- 至少 2GB 可用磁盘空间

---

## 🛠️ 环境配置

### 1. 安装 Flutter SDK

#### Windows
```powershell
# 下载 Flutter SDK
git clone https://github.com/flutter/flutter.git -b stable C:\flutter-sdk

# 添加环境变量
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\flutter-sdk\bin", "User")

# 验证安装
flutter doctor
```

#### macOS
```bash
# 下载 Flutter SDK
git clone https://github.com/flutter/flutter.git -b stable ~/flutter

# 添加环境变量
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
source ~/.zshrc

# 验证安装
flutter doctor
```

### 2. 安装 Android SDK

1. 下载并安装 **Android Studio**
2. 打开 Android Studio，进入 **SDK Manager**
3. 安装以下组件：
   - Android SDK Platform (API 21-34)
   - Android SDK Build-Tools
   - Android SDK Command-line Tools
   - Android Emulator (可选)

4. 配置环境变量：
```powershell
# Windows
[Environment]::SetEnvironmentVariable("ANDROID_HOME", "C:\Users\你的用户名\AppData\Local\Android\Sdk", "User")
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";%ANDROID_HOME%\platform-tools", "User")
```

### 3. 验证环境

```bash
flutter doctor
```

确保所有检查项都通过（绿色对勾）。

## 📦 项目配置

### 1. 克隆/打开项目

```bash
# 进入项目目录
cd changji_app

# 安装依赖
flutter pub get
```

### 2. 配置 Android 项目

#### 检查 AndroidManifest.xml
确保文件位于 `android/app/src/main/AndroidManifest.xml`，包含以下权限：

```xml
<uses-permission android:name="android.permission.INTERNET"/>
<uses-permission android:name="android.permission.RECORD_AUDIO"/>
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.CAMERA"/>
<uses-permission android:name="android.permission.FOREGROUND_SERVICE"/>
<uses-permission android:name="android.permission.POST_NOTIFICATIONS"/>
```

#### 配置 local.properties
创建 `android/local.properties` 文件：

```properties
flutter.sdk=C:\\flutter-sdk
sdk.dir=C:\\Users\\你的用户名\\AppData\\Local\\Android\\Sdk
```

### 3. 生成 Drift 数据库代码

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

如果命令失败，可以手动创建 `lib/data/database/app_database.g.dart` 文件（已包含在项目中）。

## 🔧 常见问题解决

### 1. Gradle 构建失败

**问题**：Gradle 版本不兼容

**解决**：
```bash
# 清理构建缓存
flutter clean

# 重新获取依赖
flutter pub get

# 进入 Android 目录执行
 cd android
./gradlew clean
./gradlew build
```

### 2. 依赖冲突

**问题**：某些包版本冲突

**解决**：
```bash
# 更新依赖
flutter pub upgrade

# 或手动修改 pubspec.yaml 中的版本号
```

### 3. Drift 代码生成失败

**问题**：build_runner 执行失败

**解决**：
```bash
# 删除生成的文件
rm lib/data/database/app_database.g.dart

# 重新生成
flutter pub run build_runner build --delete-conflicting-outputs

# 或持续监听模式
flutter pub run build_runner watch
```

### 4. 权限问题

**问题**：Android 权限未正确配置

**解决**：
1. 检查 `android/app/src/main/AndroidManifest.xml`
2. 确保所有需要的权限都已声明
3. 对于 Android 6.0+，需要动态申请权限（代码已实现）

### 5. 录音功能异常

**问题**：无法录音或录音文件保存失败

**解决**：
1. 检查麦克风权限是否已授权
2. 检查存储权限是否已授权
3. 查看日志：`flutter logs`

## 🚀 发布准备

### 1. 配置签名

创建 `android/key.properties` 文件：

```properties
storePassword=你的密钥库密码
keyPassword=你的密钥密码
keyAlias=你的密钥别名
storeFile=密钥库文件路径.jks
```

### 2. 更新版本号

修改 `pubspec.yaml`：

```yaml
version: 1.0.0+1  # 格式：版本号+构建号
```

### 3. 构建发布版本

```bash
flutter build apk --release
```
