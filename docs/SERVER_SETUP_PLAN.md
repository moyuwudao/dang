# 畅记云服务 - 服务器搭建需求文档及工作计划

## 一、项目概述

### 1.1 项目目标
搭建畅记 App 云端服务，提供用户登录、订阅管理、API Key 分发三项核心功能。服务端不做 AI 处理，所有 AI 调用由客户端直接完成。

### 1.2 服务器信息
| 项目 | 配置 |
|------|------|
| IP 地址 | 101.133.238.249 |
| 操作系统 | Ubuntu 22.04.5 LTS |
| CPU | 2 核 |
| 内存 | 1.6 GB |
| 磁盘 | 40 GB ESSD |
| 带宽 | 峰值 200 Mbps |
| 公网 IP | 1 个 IPv4 |

## 二、技术要求

### 2.1 技术栈
| 层级 | 技术 | 版本 |
|------|------|------|
| 运行环境 | Node.js | 20.x LTS |
| 后端框架 | NestJS | 10.x |
| 数据库 | PostgreSQL | 16 |
| 缓存 | Redis | 7 |
| 进程管理 | PM2 | 5.x |
| 反向代理 | Nginx | 1.18 |
| 容器化 | Docker | 29.x |

### 2.2 服务端功能模块
| 模块 | 功能 | 接口数 |
|------|------|--------|
| Auth | 注册/登录/Token/验证码/密码管理 | 10 |
| Subscription | 套餐查询/订阅状态/余量管理 | 4 |
| ApiKey | API Key 池管理/分配/刷新 | 2 |
| Payment | 订单/支付回调（预留） | 7 |
