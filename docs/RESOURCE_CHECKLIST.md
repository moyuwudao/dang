## 八、风险与备选方案

| 风险 | 影响 | 备选方案 |
|------|------|---------|
| ICP 备案周期长 | 延迟上线 | 先用自有 API 模式发布，备案完成后开启云服务 |
| 阿里云语音成本超预期 | 亏损 | 接入多家 ASR 服务商，动态选择最低价 |
| OpenAI 被封 | 服务中断 | 主备：DeepSeek + 通义千问 |
| 支付审核不通过 | 无法收款 | 先用第三方收款平台（如 Ping++）过渡 |
| 用户增长超预期 | 资源不足 | 阿里云弹性扩容，数据库读写分离 |
