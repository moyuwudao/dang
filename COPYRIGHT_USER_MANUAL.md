## 2. 软件概述

### 2.1 软件简介

畅记是一款基于人工智能的语音笔记应用，旨在帮助用户快速记录、整理和分析语音内容。通过集成先进的语音转写技术和AI分析能力，畅记让笔记记录变得更加高效和智能。

### 2.2 主要功能

| 功能模块 | 功能描述 |
|---------|---------|
| 语音记录 | 支持高质量录音，实时显示波形 |
| 实时转写 | 将语音实时转换为文字 |
| AI分析 | 利用AI对笔记内容进行智能分析 |
| OCR识别 | 将图片中的文字提取出来 |
| 工具台 | 提供多种AI工具，如总结、待办事项等 |
| 数据管理 | 支持笔记的分类、搜索和导出 |

### 2.3 运行环境

- **操作系统**：Android 8.0 及以上
- **存储空间**：至少 100MB 可用空间
- **网络要求**：需要联网使用AI转写和分析功能

## 3. 安装与配置

### 3.1 安装步骤

1. 下载畅记APK安装包
2. 在Android设备上安装APK
3. 启动应用程序
4. 首次打开时，系统会引导进行API配置

### 3.2 API配置

首次使用需要配置AI服务：

1. 在首页点击右上角设置图标
2. 选择"API配置"选项
3. 选择AI服务提供商（如阿里云通义千问、OpenAI等）
4. 输入API Key
5. 保存配置

### 3.3 配置说明

| 配置项 | 说明 |
|-------|------|
| API Key | 用于访问AI服务的密钥，需从服务提供商获取 |
| 服务提供商 | 选择使用的AI服务，支持多种主流AI平台 |
| 自定义URL | 可选配置，用于私有部署的AI服务 |

## 4. 基本操作指南

### 4.1 首页概览

首页主要包含以下元素：

- **顶部导航栏**：搜索、收藏、工具台、设置按钮
- **笔记列表**：按时间顺序显示所有笔记
- **快捷操作按钮**：快速创建文本笔记、拍照识别、语音记录

### 4.2 创建语音笔记

1. 点击首页底部的麦克风按钮
2. 进入录音页面后，点击中间的圆形按钮开始录音
3. 录音过程中会显示波形和时长
4. 再次点击圆形按钮停止录音
5. 录音完成后自动进入转写流程

### 4.3 创建文本笔记

1. 点击首页底部的文本按钮
2. 在弹出的对话框中输入笔记内容
3. 点击保存完成创建

### 4.4 创建OCR笔记

1. 点击首页底部的相机按钮
2. 拍摄或选择图片
3. 系统自动识别图片中的文字
4. 确认识别结果并保存

## 5. 功能详解

### 5.1 录音功能

**功能特点**：
- 支持高质量WAV格式录音
- 实时波形显示
- 录音时长显示
- 支持录音取消

**操作步骤**：
1. 进入录音页面
2. 点击录音按钮开始
3. 点击停止按钮结束
4. 录音文件自动保存

### 5.2 转写功能

**功能特点**：
- 支持实时转写
- 支持多种AI服务
- 自动分段处理长音频
- 转写结果可编辑

**操作步骤**：
1. 创建录音笔记后，系统自动触发转写
2. 等待转写完成
3. 查看转写结果
4. 如需修改，点击编辑按钮

### 5.3 AI分析功能

**功能特点**：
- 支持多种分析角色
- 智能总结笔记内容
- 生成待办事项
- 提取关键信息

**操作步骤**：
1. 打开笔记详情页
2. 点击AI分析按钮
3. 选择分析角色
4. 等待分析结果

### 5.4 OCR功能

**功能特点**：
- 支持拍照识别
- 支持图片选择
- 多语言识别
- 识别结果可编辑

**操作步骤**：
1. 进入OCR页面
2. 拍摄或选择图片
3. 等待识别完成
4. 确认并保存结果

## 7. 数据管理

### 7.1 数据存储

- 本地SQLite数据库存储笔记信息
- 录音文件存储在应用私有目录
- 支持数据备份和恢复

### 7.2 导出功能

**支持导出格式**：
- TXT文本格式
- JSON格式
- 音频文件（WAV格式）

**操作步骤**：
1. 打开笔记详情页
2. 点击分享按钮
3. 选择导出格式
4. 选择保存位置

### 7.3 删除与恢复

**删除流程**：
1. 在笔记详情页点击删除按钮
2. 确认删除操作
3. 笔记移至回收站

**恢复流程**：
1. 在设置中进入回收站
2. 选择要恢复的笔记
3. 点击恢复按钮

**永久删除**：
- 回收站中的笔记保留7天
- 7天后自动永久删除
- 可手动永久删除

## 8. 常见问题解答

### Q1：转写功能需要联网吗？

A：是的，转写功能需要连接互联网，因为需要调用云端AI服务进行语音转文字处理。

### Q2：录音文件保存在哪里？

A：录音文件保存在应用的私有目录中，其他应用无法直接访问。

### Q3：如何备份数据？

A：在设置页面中找到"备份管理"选项，可以将数据导出备份。

### Q4：支持离线使用吗？

A：录音功能可以离线使用，但转写和AI分析需要联网。

### Q5：笔记支持分享吗？

A：支持，可以将笔记内容分享为文本或文件。
